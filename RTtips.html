RawTherapee Workflow Efficiency

Master the "Do This First" strategy to automate your raw processing.

1 The Custom Profile Setup

To stop repeating tedious adjustments for every photo, create a s.pp3 profile. This profile should be set as your Global Default in Preferences > Image Processing > Default Processing Profile.

1. Color Management

  • • Go to the Color Tab.
  • • Under Input Profile, select Custom.
  • • Browse and select your specific camera's ICC/DCP profile.
  • • Pro Tip: Create a Custom Profile for each camera that you own.

2. Exposure & Tone

  • • Go to the Exposure Tab.
  • • Enable Auto Matched Tone Curve. This analyzes the embedded JPEG to give you a natural starting point.
  • • Set Exposure Compensation to 0 (neutral).

3. Raw / Capture Sharpening

  • • Go to the Raw Tab.
  • • Enable Capture Sharpening.
  • • This sharpens at the demosaicing stage, which is far more efficient and cleaner than standard sharpening.

4. Lens Profile

  • • Go to the Transform Tab.
  • • Under Lens Profile Correction, choose Automatic.
  • • Ensure "Read lens profile from metadata" is selected to let RT pull data from the EXIF.

How to Save:

Once these are set, click the Save Profile icon at the top right, name it start.pp3 (or use the model of your camera) and set it as your default in Preferences.

2 Pro Tips & "Workflow Hacks"

🚀 The 1:1 Rule

Many modules (like Noise Reduction and Sharpening) only show their true effect at 100% zoom. Don't waste time tweaking them at "Fit to Screen" view; always use the 1:1 icon.

🛠 The "Crop Blackout" Hack

By default, cropped areas are visible. Click the Black/Gray/White icons in the top toolbar to hide the cropped-out parts. This makes judging composition much easier.

🎨 Lab Mode for "Pop"

Use the L*a*b* Adjustments panel at the bottom of the Exposure tab. Increasing Chromaticity here adds color without shifting hues, which is safer than using the Standard Saturation slider.

⚡ Fast Queueing

Don't wait for images to export. Hit Ctrl+B (or the Gear icon) to send your current edit to the Queue and immediately move to the next photo. Process them all in a batch at the end.
