Real-Time Synchronization (Sync)

The File Browser Sync is the fastest way to apply changes to multiple images at once. Unlike opening an editor, you make adjustments directly in the File Browser tab.

  • Select Your Range: Use Shift or Ctrl to select multiple thumbnails.
  • The Right Panel: Adjust any tool in the Batch Edit panel on the right side of the screen.
  • Incremental vs Absolute: In "Add" mode, a slider change adds to the existing value. In "Set" mode, it forces every photo to that exact value.

Copy, Paste, & Partial Profiles

The surgical approach to batch editing. Edit one photo to perfection, then distribute those settings selectively.

Copy Profile

Right-click a processed thumbnail and choose Processing Profile Operations > Copy. This stores the current PP3 settings in your clipboard.

Full Paste

Select target images, right-click, and select Paste. This completely overwrites the target images' settings with your copied profile.

Partial Paste

The pro choice: Only paste specific tools (e.g., just the Tone Curve or Sharpening) while leaving other edits like Exposure untouched.
